# leet-code-tdd

> leet-code-tdd

个人建议：不看任何别人给的思路，全部靠自己解题
## 关于刷leetcode

个人还是很反感刷题的，尤其是这种功利心太强的刷题

但是目前站在大厂企业的角度讲

1. 很难站在客观的角度去考察一个人的能力

2. leet-code 在一些情况下可以过滤掉一批脑子不太好使的程序员

目前觉得以tdd的节奏开始刷leetcode是比较合适

## 快速开始

```shell
git clone git@github.com:wuhaohao1234/leet-code-tdd.git

cd leet-code-tdd

yarn

yarn dev
```
